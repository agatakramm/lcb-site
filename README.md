# Local Contact Backup Site

Static website for the **Local Contact Backup** Android app.

This repository contains the public contact page and Privacy Policy pages published for the app at:

**https://lcb.jejx.xyz/**

## Purpose

The site provides:

- a simple public landing page for the app
- contact information for privacy-related inquiries
- Privacy Policy pages in multiple languages

## Files

- `index.html` - main contact page
- `privacy.html` - Privacy Policy in English
- `privacy-pl.html` - Privacy Policy in Polish
- `privacy-fr.html` - Privacy Policy in French
- `privacy-de.html` - Privacy Policy in German
- `privacy-es.html` - Privacy Policy in Spanish
- `privacy-pt.html` - Privacy Policy in Portuguese
- `style.css` - shared stylesheet
- `lcb.png` - app icon used on the site
- `lcb.ico` - favicon
- `CNAME` - custom domain configuration for GitHub Pages

## Publishing

The site is published with **GitHub Pages** and uses the custom domain:

**lcb.jejx.xyz**

## Notes

This repository contains only the static website for the app.  
It is separate from the Android application source code.
