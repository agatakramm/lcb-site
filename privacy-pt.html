Local Contact Backup

EN · PL · FR · DE · ES · PT

Esta Política de Privacidade explica como o aplicativo Local Contact Backup trata os dados do usuário. O app exporta seus contatos para um arquivo CSV localmente no seu dispositivo.

Última atualização

14 de março de 2026

1. Quem somos

Este aplicativo (“Local Contact Backup”) é fornecido por Jejx.

Contato: https://lcb.jejx.xyz/

2. Quais dados o app acessa

O app acessa os contatos do dispositivo apenas quando você inicia uma exportação. Os dados exportados podem incluir os campos de contato disponíveis no seu aparelho (por exemplo: nomes, números de telefone, e-mails, endereços, notas ou outros campos relacionados).

Permissão utilizada: READ_CONTACTS (somente para a exportação solicitada por você).

3. Como os dados são usados

O app usa os contatos exclusivamente para gerar um arquivo CSV de backup solicitado pelo usuário.

  • O app não usa os contatos para publicidade.
  • O app não usa os contatos para marketing.
  • O app não vende dados de contatos.

4. Armazenamento e transferência

Os backups são exportados como CSV e salvos localmente no local escolhido por você, usando o Storage Access Framework (SAF) do Android.

O app não envia nem transmite seus contatos para qualquer servidor.

5. Compartilhamento de dados

O app não compartilha dados de contatos com terceiros como parte de sua função principal.

Se você usar o recurso de compartilhamento, o arquivo CSV exportado será compartilhado apenas com os apps que você selecionar explicitamente no diálogo de compartilhamento do Android. Esses apps podem ter suas próprias políticas de privacidade.

6. Controle do usuário

Você decide se cria um backup, onde salvá-lo e se deseja compartilhar o arquivo exportado.

Você é responsável pela segurança do arquivo exportado.

7. Serviços de terceiros

Na versão atual, o app não utiliza serviços de análise, SDKs de publicidade ou servidores remotos.

8. Segurança

O app usa mecanismos padrão do Android (SAF) para salvar e compartilhar o arquivo mediante ação explícita do usuário. Você é responsável por manter o arquivo exportado seguro.

9. Privacidade de crianças

O app não é direcionado a crianças e não coleta intencionalmente dados pessoais de crianças.

10. Alterações desta política

Podemos atualizar esta política. A versão atualizada será publicada nesta página com uma nova data.
