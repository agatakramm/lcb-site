Privacy Policy

Local Contact Backup

This Privacy Policy explains how the Local Contact Backup application handles user data.

Effective Date

March 14, 2026

Overview

Local Contact Backup is an Android application designed to help users export and save contact data locally on their device.

The app is intended to operate locally, for the purpose of creating contact backups initiated by the user.

Data Access

In order to create a backup file, the app may access contact data stored on the user's device, but only when the user chooses to perform such an action.

This contact data may include information saved by the user in their device contacts, such as names, phone numbers, email addresses, postal addresses, notes, or other contact-related fields available on the device.

How Data Is Used

The app uses contact data solely to generate a backup file requested by the user.

  • The app does not use contact data for advertising purposes.
  • The app does not use contact data for marketing purposes.
  • The app does not sell contact data.

Data Storage and Transfer

Backup files created by the app are saved locally by the user, using the location chosen by the user on their device or in their selected storage destination.

The app is intended to create local backups rather than upload contact data to a remote server as part of its core function.

Data Sharing

The app does not share contact data with third parties as part of its core backup function.

If the user chooses to manually share an exported backup file, that action is performed by the user and is outside the automatic operation of the app.

User Control

The user decides whether to create a backup and where to save it.

The user also remains responsible for any backup files they keep, copy, move, or share after export.

Children's Privacy

The app is not specifically directed to children.
